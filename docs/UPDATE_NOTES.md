## 2026-07-21 (화) — v0.9.0-beta2

- **DRM 보호 문서 인덱싱 지원**: `File is not a zip file` / `Package not found`로 실패하던 DRM 보호 xlsx/pptx/docx를 Office를 통해 자동으로 열어 인덱싱하도록 수정
- **유휴 감지 정확도 개선**: 이전엔 사용자가 작업 중이어도 일정 주기마다 무조건 인덱싱이 돌던 버그를 고쳐, 실제 자리 비움 여부를 인식하도록 함
- 자리를 8분 이상 비우면 DRM 세션 만료로 보고 DRM 의심 문서만 건너뛰고, 자리로 복귀하면 즉시 캐치업 인덱싱 실행
- **Office 점유 가드 자기 감지 버그 수정**: 인덱싱이 COM으로 직접 띄운 Office(WINWORD.EXE 등)를 스스로 "사용자 점유"로 오판해 뒷부분 문서를 계속 건너뛰던 문제 해결 + 좀비 프로세스 정리
- **앱 중복 실행 방지**: 이미 실행 중일 때 다시 실행하면 새 창 대신 기존 창이 자동으로 복원되도록 함(데이터 손상 예방)
- 메일 인덱스 스키마 마이그레이션 오류(`mail_date_ts`) 자동 복구
- 큐 처리 우선순위 로직이 실제로는 적용되지 않던 잠재 버그 수정 + COM 경유 문서 우선 처리
